
# lerna-yarn-react
Project ReactJS mono repo

## Demo projects made of 

  - Component Libraries
  - UI-Appication
  - Utils libraries
  

### ui-components:
 Codebase for reusability of components.

### common-utils:
 These are common to any project.

### product-one:
 Sample react application

### Preparation
```sh
lerna bootstrap
```

### Run ui-app :
```sh
yarn start
```
Open on browser: http://localhost:3000/
